<h1>A simple Room Booking API.</h1>
  <p>This project demonstrates how to implement Role Based Access Control using jwt for access control, Gorilla mux for routing, gorm as an orm engine and my sql as database.</p>
<h2>Requirements, Build and Run</h2>
<p>Requirements include:-</p>
<p>my sql and go installed</p>
  <h2>Build and Run</h2>
  <p>Copy this repository.</p>
  <p>Run the main.go file.</p>
  <p>Access via localhost</p>
