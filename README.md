<h1>Simple Room Booking API.</h1>
  <p>A simple Room Booking api to demonstrates Role Based Access Control.</p>

  <h2>Components</h2>
  <p>The project is written in Go and uses;</p>
  <ul>
    <li><b>Gorilla mux</b> - for routing.</li>
    <li><b>jwt</b> - for access control implementation.</li> 
    <li><b>Go ORM</b> - as an orm engine.</li>
    <li><b>mySQL</b> - to store data.</li>
  </ul>
